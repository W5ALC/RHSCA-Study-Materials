**KYWRD:** 001b
**RHCSA Objective: **001b Use input-output redirection

**Summary**
This article will demonstrate several examples of suing input-output redirection. This article discusses [[+input redirection]]. This article discusses [[+output redirection]].

**001b - examples:**
* [[+001b - example 1]] : Open a terminal on a CentOS/RHEL machine, echo "Hello World", and redirect it to a file called hello_world.txt.
* [[+001b - example 2]] : Get a list of all mounted filesystems, and send it to a file named mounted_filesystems_jan2029.txt.
* [[+001b - example 3]] : Create an unsorted list of tech companies, sort the companies alphabetically, and create a file that contains the sorted output.
