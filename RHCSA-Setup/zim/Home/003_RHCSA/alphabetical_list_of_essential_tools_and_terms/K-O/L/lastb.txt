**RHCSA Objective: **001a Access a shell prompt and issue commands with correct syntax

**Summary:**
The **lastb** command displays a list of the lastb users who logged in to a particular Linux/Unix system.  It displays this list by searching the **/var/log/btmp** file. In addition to the last users who logged into the system, the lastb command also shows bad login attempts.

**lastb:**
* lastb - function: display a list of the last users logged into a system, and bad login attempts.
