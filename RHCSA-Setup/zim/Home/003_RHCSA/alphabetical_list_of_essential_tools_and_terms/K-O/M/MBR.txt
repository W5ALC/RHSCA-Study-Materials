**Summary**
MBR stands for Master Boot Record. The MBR is located in the first physical sector of a bootable storage device, such as a hard drive. The MBR contains computer code needed for the Home:RHCSA:alphabetical list of essential tools and terms:A-E:B:Boot Loader to load the operating system kernel into memory.
