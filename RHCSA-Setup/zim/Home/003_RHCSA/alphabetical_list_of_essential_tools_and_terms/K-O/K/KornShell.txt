**RHCSA Objective: **001a Access a shell prompt and issue commands with correct syntax

**Summary:**
The **KornShell** is a command-line interpreter for Linux/Unix that was developed by David Korn at Bell Labs in the early 1980s.  The KornShell was initially based on the source code for the Bourne shell.
