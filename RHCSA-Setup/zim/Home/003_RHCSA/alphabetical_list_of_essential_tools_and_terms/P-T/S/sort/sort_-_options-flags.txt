**Summary**
This article lists the options available with the [[sort]] command.

**KEYWORD - options:**
* -b, --ignore-leading-blanks: ignore leading blanks.
* -d, --dictionary-order: considers blanks and alpha-numeric characters only.
* -f, --ignore-case: fold lower case to upper case characters.
* -g, --general-numeric-sort: compare according to general numeric values.
* -i, --ignore-nonprinting: consider only printable characters.
* -M, --month-sort: sort by month e.g. Jan < Feb < Mar.
* -h, --human-numeric-sort: sort human-readable numbers e.g. 2K < 1M < 1G.
* -n, --numeric-sort: sort according string numerical value e.g. one < two < three.
* -R, --random-sort
* --random-source=//FILE//
* -r, --reverse: sort in reverse order e.g. descending vice ascending.
* --sort=//WORD//
* -V, --version-sort
