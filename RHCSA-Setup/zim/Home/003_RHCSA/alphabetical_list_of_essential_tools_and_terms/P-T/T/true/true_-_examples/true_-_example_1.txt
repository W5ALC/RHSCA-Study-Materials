**true - tasks **(command line commands are __highlighted__)**:**
* Start your CentOS VM using the Virtual Machine Manager.
* Login as a non-root user.
* From the command line, issue the following command: __true;echo $?__
* Note that the [[Home:003 RHCSA:alphabetical list of essential tools and terms:P-T:T:true|true]] command exits with an exit code of 0, which means "SUCCESS".
* From the command line, issue the following command: __false;echo $?__
* Note that the [[Home:003 RHCSA:alphabetical list of essential tools and terms:F-J:F:false|false]] command exits with an exit code of 1, which means "FAILURE".
