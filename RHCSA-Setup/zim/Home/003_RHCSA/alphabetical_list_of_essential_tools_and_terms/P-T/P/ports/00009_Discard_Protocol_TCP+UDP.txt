**RHCSA Objective: **002h Start, stop, and check the status of network services

**Summary:**
The Discard Protocol is a null service used for connection testing and debugging. It uses **TCP/UDP ports 9**.

**Relevant Discard Protocol notes:**
* Dicard Protocol is TCP/UDP equivalent of Unix filesystem node **/dev/null**.
* Dicard Protocol on Unix-like systems is built into the **inetd** or **xinetd** daemons. Neither was installed on my Linux system at the time of this writing.
