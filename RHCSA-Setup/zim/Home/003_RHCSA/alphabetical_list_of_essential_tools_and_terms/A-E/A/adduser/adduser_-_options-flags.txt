**KYWRD:** adduser
**RHCSA Objective: **006a Create, delete, and modify local user accounts

**Summary**
This article details the available options for the adduser command.

**adduser - options:**
* -b, --base-dir BASE_DIR: 
	* If id HOME_DIR is not specified, then the default base directory for the system is used.
	* BASE_DIR is concatenated with the account name to define the home directory.
	* If the -m option is not used, BASE_DIR must exist.
