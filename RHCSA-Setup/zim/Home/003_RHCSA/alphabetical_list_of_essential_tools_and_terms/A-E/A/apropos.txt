**KYWRD:** apropos
**RHCSA Objective: **001k Locate, read, and use system documentation including man, info, and files in usrsharedoc

**Summary**
The Linux apropos command is used to search the manual pages names and descriptions for keywords and regular expressions.

**apropos:**
* apropos - function: search the manual page names and descriptions.
* apropos - description: Each manual page comes with a short description. The apropos utility searches these descriptions, and the man page titles,  for given keywords or regular expressions.
