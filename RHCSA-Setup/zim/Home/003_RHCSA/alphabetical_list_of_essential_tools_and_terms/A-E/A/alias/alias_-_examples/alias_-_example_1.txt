**KYWRD:** alias - example 1
**RHCSA Objective: **001a Access a shell prompt and issue commands with correct syntax

**Summary**
List all of the aliases currently in use on a Linux system.

**Scenario**
G'Lo is a RHEL syadmin in Reactor Department onboard the U.S.S. Theodore Roosevelt. He logs into his Linux system, and wants to see all of the aliases that are in use. What command will he run?

**alias - example 1 - tasks **(command line commands are __highlighted__)**:**
1. Startup your CentOS/RHEL system, and login as a normal user.
2. List all aliases in use using the alias command: __alias__
3. Send the output of the alias command to a file named aliases.txt: __alias > aliases.txt__
4. View the contents of aliases.txt: __cat aliases.txt__
