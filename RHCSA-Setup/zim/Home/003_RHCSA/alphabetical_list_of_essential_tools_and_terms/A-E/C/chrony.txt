**KYWRD:** chrony
**RHCSA Objective: **005h Configure a system to use time services

**Summary:**
**chrony **is an implementation of the Network Time Protocol (NTP).  It is often used so synchronize the local time on a Linux/Unix machine to network time provided by a remote NTP server.  chrony can also be used as an NTP server which provides network time for remote clients.
