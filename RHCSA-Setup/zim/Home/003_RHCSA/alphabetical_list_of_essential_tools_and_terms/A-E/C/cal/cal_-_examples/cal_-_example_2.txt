**KYWRD:** cal - example 2
**RHCSA Objective: **001a Access a shell prompt and issue commands with correct syntax

**Summary:**
cal - example 2 demonstrates how to properly use the **cal** command.

**Scenario:**
Brody is a RHEL sysadmin at Facebook. He opens a terminal on his RHEL machine. He wants to display the previous month, the current month, and the next month.  The current day should be highlighted. What command will he issue?

**cal - example 2 - tasks **(command line commands are __highlighted__)**:**
1. Startup your CentOS/RHEL system, and login as a normal user.
2. At a command prompt, issue this command: __cal -3__
