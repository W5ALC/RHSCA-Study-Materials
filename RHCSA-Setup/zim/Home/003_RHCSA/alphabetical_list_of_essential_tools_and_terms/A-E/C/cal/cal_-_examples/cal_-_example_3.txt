**RHCSA Objective: **001a Access a shell prompt and issue commands with correct syntax

**Summary:**
cal - example 3 demonstrates how to properly use the **cal** command.

**Scenario:**
T.D. is a RHEL sysadmin at Global Ministries. At a command prompt, he wants to issue a command that will display a calendar of the entire current year. What command will he issue?

**cal - example 3 - tasks **(command line commands are __highlighted__)**:**
1. Startup your CentOS/RHEL system, and login as a normal user.
2. At a command prompt, issue this command: __cal -y__
