**Summary**
1G-blocks refers to the filesystem size units returned by the df -BG command. 1G-blocks means that each block unit is equal to 1024^3 bytes.  1024^3 bytes = 2^30 bytes = 1,073,741,824 bytes = 1 Home:RHCSA:alphabetical list of essential tools and terms:A-E:D:df:df - examples:df - example 1:F-J:G:GB = 1 Home:RHCSA:alphabetical list of essential tools and terms:A-E:D:df:df - examples:df - example 1:F-J:G:GiB.
