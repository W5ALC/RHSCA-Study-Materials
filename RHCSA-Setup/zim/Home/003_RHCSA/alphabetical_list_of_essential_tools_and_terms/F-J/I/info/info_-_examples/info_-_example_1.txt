**KYWRD:** info
**RHCSA Objective: **001k Locate, read, and use system documentation including man, info, and files in usrsharedoc

**Summary**
View the [[info]] page for the [[Home:003 RHCSA:alphabetical list of essential tools and terms:U-Z:W:whereis|whereis]] command, and save it as a text file.

**Scenario**
Victor is a RHEL sysadmin at the U.S. Naval Academy. He recently discovered the info command as an alternative to man pages. He wants to:
a.) Save the info page for the whereis command as whereis.txt.
b.) View whereis.txt

whereis.txt will be in Victor's home directory.

**info - example 1 - tasks **(command line commands are __highlighted__)**:**
1. Startup your CentOS/RHEL system, and login as a normal user.
2. View the [[info]] page for the [[Home:003 RHCSA:alphabetical list of essential tools and terms:U-Z:W:whereis|whereis]] command, and save it as a text file, whereis.txt:
	a. __info whereis > whereis.txt__
3. View whereis.txt: __cat whereis.txt__
