**KYWRD:** info
**RHCSA Objective: **001k Locate, read, and use system documentation including man, info, and files in usrsharedoc

**Summary**
The info commmand is used to find out information about other commands. The basic format is this:
info //[command_name]//

For examples, this command would return information about the whereis command:
__info whereis__

info is an alternative to the man command.
