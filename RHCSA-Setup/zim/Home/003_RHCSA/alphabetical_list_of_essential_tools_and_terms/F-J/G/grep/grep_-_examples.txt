This article contains examples that illustrate the correct use of the **grep** command.

**grep - examples:**
* [[+grep - example 1]] : Search an unsorted list to find all of the CEO's who assumed their positions in 2014.
* [[+grep - example 2]] : Search for all lines that contains a capital G in a list of names/phone numbers.
* [[+grep - example 3]] : Search a text file for lines that don't match a given pattern. Get some additional experience with vi, sort, pipes and output redirects.
* [[+grep - example 4]]: Have grep print the number of matched lines in a given text file.
* [[+grep - example 5]]: grep LibreOffice Impress presentation.
