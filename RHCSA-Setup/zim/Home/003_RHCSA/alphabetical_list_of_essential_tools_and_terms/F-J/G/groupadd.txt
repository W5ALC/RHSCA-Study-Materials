**KYWRD:** groupadd

**Summary**
The groupadd utility is used to add groups on Linux systems.

**groupadd:**
* groupadd - function: create a new group.
* groupadd - description: The groupadd command is used to create group accounts.  It has several options which can be used for additional functions while adding groups.
* [[+groupadd - options-flags]] 
* [[+groupadd - examples]]

**Relevant groupadd notes**
* The following files store groupadd information:
	* [[/etc/group]] : group account information.
	* [[/etc/gshadow]] : secure group account information.
	* [[/etc/login.defs]] : shadow password suite configuration.
