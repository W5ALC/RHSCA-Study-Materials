====== graphical user interface ======

**Summary**
The graphical user interface offers the computer user or Linux system administrator a graphical interface to programs such as Firefox, Libreoffice, Audacity and Evolution. In contrast, the Home:RHCSA:alphabetical list of essential tools and terms:A-E:C:command line interface offers a text-based interface to programs.
