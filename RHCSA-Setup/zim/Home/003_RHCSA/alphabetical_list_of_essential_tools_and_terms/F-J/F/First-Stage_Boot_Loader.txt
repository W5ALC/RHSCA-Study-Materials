====== First-Stage Boot Loader ======
Created Thursday 19 July 2018

**Summary**
The First-Stage Boot Loader is located in the a ROM chip on the computer's motherboard. It is contained in the first block of read-only memory that is actually run after the Power On Self Test (POST) successfully passes. The First-Stage Boot Loader is hardcoded into ROM, and cannot he changed by the computer operator.

More information on the First-Stage Boot Loader can be found here:
