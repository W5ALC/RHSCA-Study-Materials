**KYWRD:** uname - example 1
**RHCSA Objective: **001a Access a shell prompt and issue commands with correct syntax

**Summary**
Use **uname** to list the Linux kernel version on a Linux system.

**uname - example 1 - tasks **(command line commands are __highlighted__)**:**
1. Startup your CentOS/RHEL system, and login as a normal user.
2. Show which version of the Linux kernel is installed on your system: __uname -r__
