**Summary**
This article discusses the options associated with the who command.

**who - options:**
* -a, --all: same as -b -d --login -p -r -t -T -u
* -b, --boot: time of last sytem boot
* -d, --dead: print dead processes
* -H, --heading: print line of column headings
* -l, --login: print system loging processes
* --lookup: attempts to canonicalize hostnames via DNS
* -m: returns only hostname and user associated with stdin
* -p, --process: prints active processes spawned by init.
* -q, --count: provides all login names and number of users logged on
* -r, --runlevel: prints the current runlevel
* -s, --short: prints name, line and time only (default)
* -t, --time: prints the last system clock change
* -T, -w, --mesg: add user's message status as +, - or ?
* -u, --users: list users logged in to system
* --message: same as -T
* --writable: same as -T
* --help: displays help message and exits
* --version: displays version information and exits
